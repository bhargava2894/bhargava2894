## 🚀 Featured Project — Bill Buddy

> AI-powered bill management for Indian kirana stores

![Bill Buddy Dashboard](https://raw.githubusercontent.com/bhargava2894/Bill-Buddy/main/screenshots/dashboard.png)

**[View Project →](https://github.com/bhargava2894/Bill-Buddy)**

- 📸 Upload a bill photo → Google Gemini AI extracts vendor, amount & due date automatically
- 💸 Track vendor payments, overdue bills, payment history
- 📒 Khata — per-customer udhaar (credit) ledger with one-tap full settlement
- 📊 GST input reports + one-click PDF & Excel export
- 🌐 Multi-language: English · हिन्दी · తెలుగు

**Stack:** React 19 · Spring Boot 3 · PostgreSQL · Google Gemini AI · Framer Motion
